Summary:

A Flutter-based web and mobile interface for the **SaiSangha Chitfund App**.  
This project provides a clean dashboard and user management screens for tracking loans, payments, and user details.

---

## ✨ Features

- **Dashboard (lightweight view)**  
  - Shows each user’s name and phone number  
  - Displays total loan outstanding and amount to be paid  
  - Click on a user to navigate to their detailed profile

- **User Details Page**  
  - View user information  
  - Add new loans with amount, interest
